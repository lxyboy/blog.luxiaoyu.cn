#hadoop使用

## 多机

###各个主机上执行

scp administrator@10.10.100.42:/home/administrator/Downloads/learnjava/jdk-7u67-linux-x64.tar.gz ./

scp administrator@10.10.100.42:/home/administrator/Downloads/learnjava/hadoop-2.5.1.tar.gz ./


创建用户和组

addgroup hadoop
adduser -ingroup hadoop hduser

停用ipv6
vim /etc/sysctl.conf

```txt
# disable ipv6
net.ipv6.conf.all.disable_ipv6 = 1
net.ipv6.conf.default.disable_ipv6 = 1
net.ipv6.conf.lo.disable_ipv6 = 1
```

reboot

安装hadoop

tar xf jdk-7u67-linux-x64.tar.gz
tar xf hadoop-2.5.1.tar.gz

mv jdk1.7.0_67/ /opt/
mv hadoop-2.5.1 /opt/

编辑环境变量vim /etc/environment添加

```
JAVA_HOME="/opt/jdk1.7.0_67/"
HADOOP_PREFIX="/opt/hadoop-2.5.1"
```

编辑/etc/sudoers添加hduser
hduser ALL=(ALL) ALL

切换到hduser下
sudo -i -u hduser

sudo chown -R hduser:hadoop /opt/hadoop-2.5.1/



### 配置hadoop
hadoop配置文件/opt/hadoop-2.5.1/etc/hadoop

1. yarn-site.xml
2. core-site.xml
3. mapred-site.xml
4. hdfs-site.xml
5. ~/.bashrc

可以参照官网说明和查找当前*default*文件。
find ./ -name "*default*"
